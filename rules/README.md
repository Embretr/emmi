# Rules (avoid)

Each file here encodes something **not to do again**: anti-patterns, bad habits, or feedback the team rejected.

Personal rule files under this directory are **gitignored** (only this `README.md` is tracked). Clone the kit and add your own `*.md` here locally.

## Policy

- **Never create or edit files in this directory without explicit user approval.**
- You **may** suggest a new rule (path + one paragraph) when the same mistake appears twice or when feedback is strong.
- Prefer small, testable statements: "Avoid X because Y."

## Suggested filename pattern

`NNN-short-topic.md` (optional numbering once you have many rules).
