# Evaluators (prefer)

Each file here encodes **what good looks like**: metrics, checklists, or scoring hints for solutions.

Personal evaluator files under this directory are **gitignored** (only this `README.md` is tracked). Clone the kit and add your own `*.md` here locally.

## Policy

- **Never create or edit files in this directory without explicit user approval.**
- You **may** suggest a new evaluator when it would improve decisions (latency budget, accessibility bar, conversion step, etc.).

## Suggested filename pattern

`NNN-short-metric.md`
